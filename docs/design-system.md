# 数独应用设计系统

> 版本: 1.1
> 更新: 2026-07-09
> 框架: Flutter + Material Design 3

---

## 1. 设计原则

### 1.1 核心原则
- **清晰优先**: 游戏状态一目了然，玩家无需猜测
- **即时反馈**: 每次交互都有明确的视觉反馈
- **减少认知**: 避免过度设计，聚焦游戏本身
- **全球可用**: 多语言支持，文化无关设计

### 1.2 响应式策略
- 横向自适应: 数字键盘在窄屏下可横向滚动
- 字体缩放: 关键元素字体随尺寸动态调整
- 触控友好: 所有可点击元素最小 44×44pt

### 1.3 无障碍
- 所有图标配文字标签
- 色彩传达不依赖单一视觉通道
- 支持系统字体缩放

## 2. 色彩系统

### 2.1 主题色

#### 浅色模式
```dart
seedColor: Colors.blue
brightness: Brightness.light
```

#### 深色模式
```dart
seedColor: Colors.blue
brightness: Brightness.dark
```

### 2.2 语义色 - 单元格状态

| 状态 | 色值 | 用途 |
|------|------|------|
| 选中 | `blue[200]` | 当前选中的单元格 |
| 同数字高亮 | `blue[100]` | 与选中值相同的其他单元格 |
| 同行/列/宫 | `grey[200]` | 辅助定位相关区域 |
| 冲突 | `red[100]` | 违反规则的单元格 |
| 错误 | `red` | 填入值与答案不符 |
| 默认 | `white` | 普通单元格背景 |

### 2.3 语义色 - 难度等级

| 难度 | 色值 | Hex |
|------|------|-----|
| 简单 | `green` | #4CAF50 |
| 中等 | `orange` | #FF9800 |
| 困难 | `red` | #F44336 |
| 专家 | `purple` | #9C27B0 |

### 2.4 模式图标色

| 模式 | 图标色 | 用途 |
|------|--------|------|
| 每日挑战 | `amber` | 倒计时图标 |
| 自由模式 | `blue` | 默认状态 |
| 副本挑战 | `indigo` | 开发完成后 |
| 对战模式 | `red` | 开发完成后 |

### 2.5 状态指示色

| 状态 | 色值 | 用途 |
|------|------|------|
| 已完成 | `green` | 每日挑战已完成 |
| 进行中 | `blue` | 每日挑战进行中 |
| 未开始 | `grey` | 副本关卡未解锁 |
| 已解锁 | - | 副本关卡可玩 |

### 2.6 边框色

| 元素 | 色值 | 宽度 |
|------|------|------|
| 3×3 宫格边界 | `black87` | 3pt |
| 普通单元格边框 | `grey[400]` | 0.5pt |

### 8.2 模式卡片 (ModeCard)

#### 通用结构（100pt 高度）
```
┌──────────────────────────┐
│  [图标48pt]  标题  [状态] │ ← 左侧内容，右上状态

#### 布局规则
```

  children: [

)
```

## 9. 交互状态

### 9.1 点击反馈
- 所有可点击元素使用 `GestureDetector` 或 Material 组件
- 选中状态立即更新背景色
- 按钮按下时使用 Material 涟漪效果

### 9.2 加载状态
- 初始化中: `CircularProgressIndicator`
- 初始化失败: 错误图标 + 提示文字
- 匹配对手中: 骨架屏 + 动画

### 9.3 模式卡片状态

| 状态 | 视觉表现 | 适用模式 |
|------|----------|----------|
| 可用-未完成 | 正常显示，右侧箭头，显示副信息 | 每日、自由 |
| 可用-已完成 | 绿色对勾图标，显示"已完成" | 每日 |
| 禁用/待开发 | 灰度处理，锁图标，不可点击 | 副本、对战 |
| 点击反馈 | Material 涟漪效果 | 全部 |

## 11. 页面流程

### 11.1 启动流程
```
AppInitializer
    ↓
Lobby (大厅)
    ↓ 选择模式
┌───┴───┬───────┬──────────┐
│       │       │          │
每日  自由   副本     对战

### 11.2 自由模式流程
```
Lobby
    ↓
自由模式入口
    ↓
难度选择对话框
    ↓
SudokuGamePage
    ↓
完成 → 返回大厅 / 新游戏

### 11.3 每日挑战流程
```
Lobby
    ↓
每日挑战卡片
    ↓
检查今日状态
    ├─ 未完成 → SudokuGamePage (固定题目)
    ├─ 进行中 → SudokuGamePage (继续游戏)
    └─ 已完成 → 显示结果/排名
```

Color _getCellColor(BuildContext context) {
  if (cell.hasConflict) return Colors.red[100]!;
  if (cell.hasSameValueAsSelected && !cell.isSelected) return Colors.blue[100]!;
  if (cell.isSelected) return Colors.blue[200]!;
  if (cell.isInSameRow || cell.isInSameCol || cell.isInSameBox) return Colors.grey[200]!;
  return Colors.white;
}
```
